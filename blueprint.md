
# Lotto Number Generator

## Overview

This application generates a set of 6 unique random numbers for a lottery game, typically between 1 and 45.

## Features

*   **Number Generation:** Generates 6 unique random numbers.
*   **Display:** Shows the generated numbers to the user.
*   **Simple Interface:** A single button to generate the numbers.
